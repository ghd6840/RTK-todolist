### ToDo List using Redux ToolKit + Typescript

Redux Toolkit을 이용한 TodoList 만들기

사용된 기술

- React
- Redux-toolkit
- Typescript
- Styled-components

현재 기능

- TodoList 추가
- TodoList 삭제
- 클릭 시 Completed 처리;
- Active, Completed Filter

```bash
$ npm start
```

이후 [http://localhost:3000](http://localhost:3000)에서 접속 가능합니다.
